Tech Academy Live Project - DataScrape

DataScrape is a custom web scraping application built with Django and Beautifulsoup. The application gathers data from the web that is relevant to the user (localized weather, and personalized sports, entertainment, and tech news). Other technologies used include:

  • Selenium (Python Library)
  • Virtualenv (Python Library)
  • Visual Studio Code
  • Azure DevOps (Project Management)
  • Git (Source Control)

Specific contributions during my 2 weeks working on the development team include code to:

  1. Retrieve/display current local weather  info (temperature, humidity and date:time of update) within user's zipcode.
  2. Retrieve/display the three-day forecast of local weather within the zipcode of the user.
  3. Create a model for messages within a new application of the DataScrape project to store sender/receiver names, date, and message body text.
  4. Retrieve/display the NASA image of the day (APOD), along with its title, and descriptive text.
  5. Retrieve/display the current top 5 box office movie titles.
  6. Automatically populate the HockeyTeam model in the DataApp of the DataScrape project with all of the current NHL team names.
  7. Get/save the favorite NHL team to the user's personal profile management page.
  8. Retrieve/display the next five events for the user's favorite NHL team.
  9. Retrieve/display the logo of the user's favorite NHL team on the user's sports page.
